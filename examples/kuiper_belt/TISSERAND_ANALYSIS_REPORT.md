## Executive Summary

This analysis applies the **Tisserand parameter** (T), an invariant of gravitational motion, to identify dynamically linked populations of Kuiper Belt Objects. The Tisserand parameter is conserved during close gravitational encounters and provides a powerful tool for identifying objects that may share a common origin or have experienced encounters with a hypothetical massive perturber at 500 AU.

## Methodology

### Tisserand Parameter Formula

The Tisserand parameter relative to a massive body is defined as:

```
T = (a_p/a) + 2*sqrt((a/a_p)*(1-e²))*cos(i)
```

Where:
- **a_p** = Semi-major axis of perturbing body (500 AU)
- **a** = Semi-major axis of object (AU)
- **e** = Eccentricity of object
- **i** = Orbital inclination of object (degrees)

### Physical Interpretation

1. **Conserved Quantity**: T is nearly conserved during gravitational scattering events
2. **Dynamic Linking**: Objects with similar T values may have:
   - Shared a common origin
   - Experienced encounters with the same perturber
   - Evolved together dynamically
3. **Grouping Threshold**: Objects with ΔT < 0.5 are considered dynamically linked

## Technical Notes

### Formula Implementation

The analysis uses the standard Tisserand parameter formula converted to degrees:

```python
def calculate_tisserand(a, e, i_degrees, a_p):
    i_rad = math.radians(i_degrees)
    first_term = a_p / a
    factor = (a / a_p) * (1 - e**2)
    second_term = 2 * math.sqrt(factor) * math.cos(i_rad)
    return first_term + second_term
```
