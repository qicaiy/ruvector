## Methodology

### Resonance Calculation

For an object in n:1 mean motion resonance with a planet:
- **Formula**: a_resonant = a_planet / (n)^(2/3)
- **Interpretation**: Object completes n orbits for every 1 planet orbit
- **Detection Tolerance**: 3.0 AU (standard for distant object surveys)

### Analysis Parameters

- **Planet Distance Range**: 300-700 AU (50 AU increments)
- **Resonance Orders**: n = 2, 3, 4, 5, 6, 7, 8, 9
- **KBO Dataset**: 21 well-characterized Trans-Neptunian Objects
- **Total Detections**: 19 resonance associations
- **Unique Objects in Resonance**: 11 of 21 (52%)

## Resonance Physics Reference

### Mean Motion Resonance Formula

For a planet at semi-major axis a_p and object at a_o in n:1 resonance:

```
n * (a_o / a_p)^(3/2) = 1  (Kepler's 3rd Law)
=> a_o = a_p / n^(2/3)
```

### Why Different n Values Matter

- **Lower n values (2:1, 3:1)**: Objects orbiting much closer to planet
- **Higher n values (8:1, 9:1)**: Objects orbiting much farther from planet
- **Resonance width**: Depends on object's eccentricity and planet's mass
- **Stability**: Higher-order resonances (7:1, 8:1, 9:1) typically more fragile

### Observational Significance

Objects in multiple resonances (like 445473 (2010 VZ98)) suggest:
1. Strong external perturbation source
2. Extended orbital chaos zone
3. Possible long-term planet migration signature
4. Complex dynamical history

## Caveats and Limitations

1. **Limited Object Sample**: Only 21 TNOs analyzed
   - Current count of known TNOs > 3,000
   - Analysis could be repeated with complete dataset

2. **Fixed Tolerance**: 3.0 AU resonance tolerance is arbitrary
   - True resonance width depends on perturber mass
   - Should vary tolerance with planet mass estimates

3. **No Orbital Integration**: Uses instantaneous orbital elements
   - Long-term resonance stability not assessed
   - Chaotic diffusion effects not modeled

4. **Excludes Secular Resonances**: Analysis focuses only on mean motion resonances
   - Other resonance types not considered
   - Could provide additional evidence

5. **No Inclination Filtering**: Mixed inclination objects
   - Planet Nine hypothesis includes inclination prediction
   - Could improve specificity with inclined planet assumption
