TECHNICAL NOTES

Method: Mean Motion Resonance Detection
├─ Formula: a_resonant = a_planet / (n)^(2/3)
├─ Basis: Kepler's 3rd Law for orbital periods
├─ For n:1 resonance: Object completes n orbits per planet orbit
└─ Detection: Tolerance-based matching (3.0 AU)

Assumptions:
├─ Single perturber (planet mass effects linearly additive)
├─ Circular/near-circular planet orbit
├─ Resonance capture was recent (objects still in resonance)
└─ No migration history considered

Limitations:
├─ Small dataset (21 KBOs vs 3000+ known TNOs)
├─ Fixed tolerance (should vary with planet mass)
├─ No orbital integration (stability not verified)
├─ Instantaneous orbital elements (long-term evolution ignored)
└─ No secular resonance analysis (different physics)
